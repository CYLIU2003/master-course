## Comparison Rule

When reviewing a PR that touches BFF routers or frontend fetch:

1. Run `python tools/benchmark_api.py` before and after
2. If any p50 increases by more than 20%, flag for review
3. If any payload size increases by more than 50KB, flag for review
4. If a summary endpoint starts returning detail fields, reject the change

## How to Run Benchmark

```bash
# Start the app
uvicorn bff.main:app --port 8000 &

# Run benchmark
python tools/benchmark_api.py

# Stop app
kill %1
```
